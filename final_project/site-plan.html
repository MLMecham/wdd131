Overview

Purpose

My purpose for this website is to provide a fun learning oppertunity for anybody seeking a challenge. The questions will range from easy to nigh impossible so all ages will be able to find something to do. The website will ask the user what topics they want the questions to be about and how many then want to answer, then the second page will be brain buster quiz.

Audience

My audience is for anybody who can read and likes challenges. There will be general trivial to riddles. Anybody can participate

Dynamic elements

Page one will have multiple select elements with many options. The JavaScript will receive the selected options from the user when they click the "Start Quiz" button. Once they click it, the JS will load a query selector to send their options to the second page of the site, the quiz. Page two will dynamially load the quiz html from the questions pool. It will take the options passed from page one, filter the questions to that criteria, and then load the questions. the submit quiz button will then be at the bottom of the second page. When clicked, it will score the users quiz and give them their results.

Content

Home page

The home page is going to be a welcoming page that is warm and inviting

The purpose of this page is to explain to the user the reason the quiz exists, and to have the user enter information to be used generate their quiz.

Welcome to the Brain Buster's Quiz!

This quiz is designed with fun in mind, but that doesn't mean these questions are easy. Each question has a difficulty score assigned to it, these scores start at 1 and end at 5. Difficulty 1 can be reliably answered by most elementary students, while difficulty 5 are tough to answer to those even with graduate degrees because of the obscure topics they cover. All questions are either True-False questions or multiple choice, so even if the question has busted you brain, just give it your best guess.

Below you may select the number of questions you want to answer, the difficutly of questions you want to answer, and the type of question. Please select to your liking and click on the start quize button below.
